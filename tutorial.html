Primitive Types

integer, floating-point, string, boolean, null, undefined, symbol

She's 13

You cannot change the value of a primitive type once it is declared.
It is immutable.
Example:
let age = 13;
age = 18;
console.log(age);
The console will log 18, but age is not modified. It is creating a new variable instead.
Strings work in the same way.
