age = 13

You cannot change the value of a primitive type once it is declared.
It is immutable. 


        Example: 
        let age = 13;
        age = 18; 
        console.log(age);

The console will log 18, but age is not modified. age = 18 is creating a new age variable instead.

Global and Local Scope

Global Scope

Variables declared outside a function become GLOBAL
Therefore, the global variable will have global scope: All scripts and functions on a webpage can access it.

Automatically Global

If a value is assigned to a variable that has not been declared, it will automatically become GLOBAL.

Local Scope

Variables declared inside a function become LOCAL to the function.
Therefore the variables can only be accessed within the function.
