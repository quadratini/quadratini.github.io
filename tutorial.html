Literals, Variables, and CONSTANT

Data Types

Primitive Types

Integer, Floating-point, String, Boolean, Null, Undefined, Symbol

She's 13, but she's primitive

You cannot change the value of a primitive type once it is declared.
It is immutable.
Example:
let age = 13;
age = 18;
console.log(age);
The console will log 18, but age is not modified. age = 18 is creating a new age variable instead.
Strings work in the same way.

Object Types

Array, Date, Regular Expression, Map and WeakMap, Set and WeakSet

In JS, almost "everything" is an object. All JS values, except primitives, are objects. Objects are mutable and their values can be changed at anytime. Objects are Variables. But objects can contain many values.
Example: var person = {firstName:"Ron", lastName:"Doe", age:22, eyeColor:"brown"};
The named values in JS objects are called properties
In the example above, the properties are firstName, lastName, age, and eyeColor. Their respective values are Ron, Doe, 22, and brown. Objects are name value pairs, similar to...

  • Associative arrays in PHP
  • Dictionaries in Python
  • Hash tables in C
  • Hash maps in Java
  • Hashes in Ruby and Perl.

Array
